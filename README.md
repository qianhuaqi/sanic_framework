# LingShu Framework / 灵枢框架

LingShu is a greenfield, independently implemented Python Web/API framework.

LingShu does not depend on Sanic, FastAPI, Flask, Django, Starlette, or any other upper-level Web framework. The archived legacy repository state creates no compatibility obligation for the new framework.

Current phase: P0 architecture consolidation. No production framework code may be implemented until the Blueprint is confirmed and frozen by the project lead.
